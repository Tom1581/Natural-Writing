# Project Handover - Natural Writing Engine (v39.0 Alpha & Omega)

Welcome to the **Natural Writing Engine**, a 55-phase technical masterpiece. This repository represents a complete, world-grade Writing Operating System focused on enterprise-scale stylistic optimization and organizational governance.

## 🏛️ Ecosystem Architecture
-   **Backend**: NestJS (TypeScript) orchestrated via an asynchronous, event-driven service layer.
-   **Database**: PostgreSQL managed via TypeORM with professional migration hardening.
-   **Frontend**: Next.js (App Router) featuring high-fidelity Framer Motion animations and Tiptap editing.
-   **Neural Nodes**: Leverages OpenAI (Multi-Model Swarm) for stylistic DNA mirroring and consensus synthesis.

## 🚀 Deployment Guide
1.  **Environment**: Configure `.env` in the backend with `OPENAI_API_KEY`, `JWT_SECRET`, and `DB_CREDENTIALS`.
2.  **Infrastructure**: Use `docker-compose up -d` to launch the PostgreSQL and PGAdmin containers.
3.  **Bootstrapping**:
    -   Run `npm install` in both `/backend` and `/frontend`.
    -   Run `npm run seed` in the backend to populate the organizations and teams.
4.  **Production**: Use `scripts/deploy.sh` for final containerization and cloud push.
